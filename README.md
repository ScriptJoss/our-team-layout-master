## What I learn

I learned to use CSS Grid again, which helped me unlock a different way of looking at layouts and get a better grasp of using Grid.

I also reinforced concepts of responsive design, handling background images with `object-fit`, and using CSS variables (custom properties) to make project maintenance easier.
